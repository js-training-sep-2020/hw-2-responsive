How to run (build) this project on your owm machine (after you clone or download this repository).

1. Check if you have Node.js and npm installed on your computer:
        node -v
        npm -v

2. If you dont't have one, install Node.js and npm for your OS from https://nodejs.org/
When you install Node.js, you automatically get npm installed.

3. Install dependencies:
        npm i

4.Run webpack plugins:
        npm start

5. Run lite-server:
        npm run dev

NOTE: to chose which version of the styles you want to render (flex or css-grid one), comment or uncomment import
of according file at 1st and 2nd lines of index.js.

Have a nice work!
